## Attività. 

Il laboratorio ha l'obiettivo di introdurre alla pubblicazione di dati geospaziali sul web e allo sviluppo di applicazioni cartografiche interattive ed è divisa in due part:

### Introduzione alla pubblicazione di dati geospaziali su web per mezzo della libreria Openlayers.
   
Verranno illustrati i concetti generali e gli oggetti principali usati per implementare viste e controlli di mappa, collegare layer, definire gli stili di rappresentazione. Verranno inoltre introtte le risorse di documentazione a disposizione degli sviluppatori.
E' necessaria una preconoscenza dei principi di base di sviluppo web (html, css, javascript) e la disponibilità di un pc con il server di sviluppo Nodejs ed un editor per programmazione (Per esempio Geany o Brackets)

### Introduzione allo sviluppo veloce di moderne applicazioni cartografiche su web usando come piattaforma il webgis wegue https://wegue-oss.github.io/wegue/#/ basato su Vuejs ed Openlayers
   
Verrà illustrato il deployment in locale dell'applicazione base (il cd "boilerplate") e la modifica dei relativi files di configurazione. Verrà poi illustrato come interagire con il sistema degli oggetti messi a disposizione della piattaforma per personalizzare l'interfaccia del webgis usando Vuetify https://vuetifyjs.com/, il modulo di componenti visuali per Vuejs usato da Wegue.
Per seguire il workshop è necessaria la preconoscenza dei concetti di base di openlayers e di sviluppo web (javascript, html, css). Si consiglia la disponibilità di pc con installazione di nodejs e editor di programmazione (Geany, Brackets o simili)
