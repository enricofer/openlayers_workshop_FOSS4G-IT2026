## Attività. 

Il laboratorio ha l'obiettivo di introdurre alla pubblicazione di dati geospaziali sul web e allo sviluppo di applicazioni cartografiche interattive utilizzando le librerie OpenLayers e WEGUE webgis.

## Prerequisiti

Per seguire con profitto il workshop è consigliabile una conoscenza di base dello sviluppo web e della sintassi dell formato HTML e del linguaggio javascript.

## Prima di iniziare

Prima di iniziare il workshop, segui le istruzioni nella pagina [Setup](setup.qmd) per configurare nodejs sulla tua macchina locale.
